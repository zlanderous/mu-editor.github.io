Informations sur Mu

{% comment %}

Mu is a Python code editor for beginner programmers based on extensive feedback given by teachers and learners.

{% endcomment %}

Mu est un éditeur de code pour programmeurs débutants basé sur beaucoup d'avis donnés par enseignants et élèves.

Télécharger Tutoriels Comment faire..?

Moins c'est plus.

Mu n'a que les fonctionnalités les plus essentielles, pour que les utilisateurs ne soient pas intimidés par une interface trop compliquée.

Emprunte le chemins de moindre résistance.

Quelle que soit la tâche, il y a toujours une façon évidente de la faire avec Mu.

Favorise la simplicité.

It's quick and easy to learn Mu ~ complexity impedes a novice programmer's first steps.

C'est rapide et facile d'apprendre Mu ~ la complexité nuit aux premiers pas d'un programmeur.

Amuse toi!

L'apprentissage devrait inspirer l'amusement ~ Mu aide les élèves à rapidement créer et tester du code fonctionnel.

La citation d'Alan Kay sur l'apprentissage de la programmation résume la vision de Mu. Mu vise à être la "vraie chose" en tant qu'environment de développement pour des programmeurs débutants faisant leurs premiers pas avec Python.
